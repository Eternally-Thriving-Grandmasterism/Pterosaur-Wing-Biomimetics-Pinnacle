# β-Keratin Nanofibril Hierarchy in Bird Barbules Lattice

MIT License - Eternally-Thriving-Grandmasterism

Hierarchy Levels:
0. Polypeptide Chain (~100–140 aa, β-sheet domains + loops, Cys disulfide bridges)
1. β-Sheet Protofibril / Nanofibril Core (2–5 nm Ø)  
   - 2–4 antiparallel β-strands → twisted ribbon  
   - Crystalline β-sheet core + amorphous surface loops
2. Microfibril (20–50 nm Ø)  
   - 8–20 protofibrils in quasi-hexagonal packing  
   - Inter-fibril disulfide + H-bonds
3. Macrofibril / IF Bundle (0.1–1 μm Ø)  
   - 10–100 microfibrils in amorphous KAP matrix
4. Barbule Cortex (2–10 μm Ø)  
   - Parallel macrofibrils + air pockets in medulla
5. Barb / Vane (10–100 μm scale)  
   - Interlocking proximal hooklets + distal nodes → rigid airfoil

Properties Gradient:
- Nano: 10–20 GPa modulus, brittle
- Micro: 5–15 GPa, tougher
- Macro/Barbule: 3–10 GPa, high toughness (1–5 MJ/m³)
- Density: ~1.2–1.3 g/cm³ (air voids reduce effective weight)

Biomimetic Mercy Mapping:
- Hierarchical β-sheet nanofibers → high-toughness aligned composites (electrospinning + β-peptides)
- Interlocking hooklet-node → reversible micro-fasteners (self-repairing panels)
- Air-pocketed macrofibrils → ultralight structural foams
- Preen oil mimic → self-lubricating, dirt-repellent coatings
